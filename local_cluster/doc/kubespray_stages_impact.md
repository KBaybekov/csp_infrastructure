| Этап                      | Что делает                                                                                                                                                                                | Уровень влияния | Пояснение                                                                                                                                                                                                          |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| bootstrap_os                  | Устанавливает Python, настраивает хостовое имя, отключает swap, проверяет DNS.                                                                | Высокий                | Меняет конфигурацию ОС: отключает swap, правит /etc/hosts, устанавливает пакеты.                                                                                      |
| system_packages               | Устанавливает базовые пакеты (apt-transport-https, ca-certificates, curl, и т.д.)                                                                                     | Средний                | Добавляет/обновляет пакеты, но не затрагивает работающие сервисы критично.                                                                                  |
| kubernetes/preinstall         | Загружает модули ядра (overlay, br_netfilter), настраивает sysctl (ip_forward, bridge-nf-call-iptables), создаёт директории для kubelet и etcd. | Высокий                | Изменяет параметры ядра и создаёт системные директории. Требует перезагрузки не всегда, но влияет на сеть и работу kubelet. |
| container_engine (containerd) | Устанавливает и настраивает containerd (или Docker), включает его сервис.                                                                             | Высокий                | Устанавливает новый сервис, который будет запускать все контейнеры. Без него кластер не работает.                                         |
| etcd                          | Генерирует сертификаты etcd, устанавливает etcd, запускает кластер etcd.                                                                         | Критический        | Создаёт распределённое хранилище данных кластера. Ошибка на этом этапе разрушает весь кластер.                                             |
| kubernetes/control-plane      | Устанавливает kube-apiserver, kube-controller-manager, kube-scheduler, генерирует сертификаты, настраивает kubeconfig.                                | Критический        | Управляющая плоскость Kubernetes. Без неё нет API.                                                                                                                                             |
| kubernetes/node               | Устанавливает kubelet, kube-proxy, настраивает CNI (обычно Calico), запускает сетевые поды.                                                      | Высокий                | Обеспечивает работу узлов кластера. Влияет на сеть и запуск подов.                                                                                                   |
| network_plugin (Calico)       | Разворачивает DaemonSet Calico, настраивает IP pools, BGP (если нужно).                                                                                           | Высокий                | Без сетевого плагина поды не могут общаться между собой.                                                                                                                     |
| kubernetes/kubeadm            | Выполняет kubeadm init на первом control-plane узле и kubeadm join на остальных.                                                                                  | Критический        | Инициализирует кластер. При ошибке кластер не создаётся.                                                                                                                    |
| addons                        | Устанавливает DNS (CoreDNS), dashboard (опционально), metrics-server и другие дополнения.                                                                 | Средний                | Не влияет на базовую работу кластера, но важны для сервисов внутри.                                                                                                  |
| post-kubernetes               | Настраивает kubectl для пользователя, копирует kubeconfig, перезапускает kubelet.                                                                   | Низкий                  | Удобство администратора, не влияет на стабильность.                                                                                                                             |
